# Tasbeeh Counter (تسبيح)

A beautifully designed digital tasbeeh counter built using HTML, CSS, and JavaScript.

## Features
- Multiple dhikr tracking (33 / 34 counts)
- Completed sets tracking
- Daily reset system
- LocalStorage persistence
- Smooth UI with Islamic theme

## Tech Used
- HTML5
- CSS3 (Responsive Design, Animations)
- Vanilla JavaScript (DOM + State Management)

## How it Works
- Each dhikr has its own counter and completed sets
- Global daily total is tracked separately
- Data is stored in localStorage and resets every 24 hours

## Future Improvements
- Dynamic rendering of beads
- Event listeners instead of inline handlers
